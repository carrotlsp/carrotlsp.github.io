  • 问：如何验证 python3 安装是否成功呢？
  • 答：输入 python3 能进入 python 指令模式即为成功。
  • 问：如何验证 numpy matplotlib opencv_python 是否安装成功呢？
  • 答：进入python模式后，分别输入 import numpy、import matplotlib、import cv2 不会报错，即安装成功。
